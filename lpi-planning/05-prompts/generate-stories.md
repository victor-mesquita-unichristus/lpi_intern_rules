# Prompt: Generate Stories

Use os documentos de entidades, fluxos e regras transversais em `lpi-planning/` para gerar histórias de backlog.

## Instruções

1. Leia os arquivos relevantes para o escopo desejado.
2. Produza histórias focadas em negócio usando o template de história.
3. Mantenha cada história coesa e valiosa de forma independente.
4. Inclua critérios de aceite explícitos derivados das regras documentadas.
5. Destaque dependências entre entidades, fluxos e regras transversais.

## Saída esperada

- Lista ordenada de histórias
- Título da história
- Declaração da história de usuário
- Critérios de aceite
- Dependências
